Text-switch

ได้รับแรงบันดานใจมาก เว็บ codepen ของคุณ Nooray Yemon จากที่ได้สักเกตเห็นว่าตัวโค้ดสามารถเอามาประยุคต์ให้สามารถเขียนได้ง่ายมากขึ้นทางทีมผู้พัฒนาจึงได้ทำการเขียนโค้ดใหม่ให้เข้าใจง่ายมากขึ้นจากเดิม

หากเพื่อน ๆ ได้สังเกตทางทีมงานยังคงใช้งาน bootstrap ใช้การช่วยในการพัฒนาทั้งนี้ด้วยเหตุผลที่ว่า จะได้ลดการเขียน css ด้วยตนเอง และคำสั่งที่ทีมงานยังเป็นคำสั่งพื้นฐานของ css

html
                    
    <div class="text-center text-switch">
        header text

        <div class="text-switch-body">
            <div class="text-switch-box">

                body text

            </div>
        </div>

        footer text
    </div>
css
                    
    .text-switch{
        width: 30%;
        margin: 0 auto;
        font-size: 3.5rem; /* same size */
    }

    .text-switch-body{
        overflow: hidden;
        height: 3.5rem; /* same size */
    }

    .text-switch-box{
        -webkit-animation: text_switch 5s infinite; /* Chrome, Safari, Opera */
        animation: text_switch 5s infinite;
        margin-top: 0;
    }

    .text-switch-body p{
        line-height: 3.5rem; /* same size */
        padding: 0;
        margin: 0;
    }

    /* Chrome, Safari, Opera */
    @-webkit-keyframes text_switch {
        0% {
            margin-top: 0;
        }
        25% {
            margin-top: -3.5rem;
        }
        75%{
            margin-top: -7.0rem;
        }
        100%{
            margin-top: 0;
        }
    }

    /* Standard syntax */
    @keyframes text_switch {
        0% {
            margin-top: 0;
        }
        25% {
            margin-top: -3.5rem;
        }
        75%{
            margin-top: -7.0rem;
        }
        100%{
            margin-top: 0;
        }
    }

จาก code ที่่แสดงข้างบน หัวใจที่ทำให้ตัว component นี้ต่างจาก component อื่นคือ การใช้คำสั่ง overflow และ animation ในการทำงานของ component โดยหลักการทำงานของ component text-switch มาจาก class .text-switch-body คำสั่ง overflow: hidden; ความหมายคือ เมื่อขนาดของวัตถุข้างในใหญ่กว่าขนาดของ .text-switch-body จะทำการแอบส่วนที่เกินเอาไว้ โดยค่าเริ่มต้นของ .text-switch-body มีส่วนสูงเท่ากับ 35px เท่ากับข้านของ line-height และ font-size เพื่อเมื่อให้การสร้าง animation ง่ายมากขึ้น
